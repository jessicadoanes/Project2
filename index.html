Welcome to the 100 years of Aviation Crashes!

This website contains visualizations of aviation crashes from 1908 up to 2009.

Get Started

Overview

This website was created to satisfy the requirements of Project 2 for GA Tech's Data Science and Analytics certifcate program. We sought to create a website with four dynamic visualizations analyzing aviation crashes from 1908 to 2009.

Project Features

This website is powered by a Python Flask–powered API, HTML/CSS, JavaScript, and at least one database (SQL, MongoDB, SQLite, etc.).

Python Flask–powered API

This site includes visuals powered by a flask-powered API from the International Civil Aviation Organization (ICAO).

HTML/CSS and JavaScript

We used HTML/CSS to create and design this website.This site displays data across four pages using various features from Bootstrap.

Data for this project was retrieved from Kaggle. The dataset contains data of airplane accidents involving civil, commerical, and military transport worldwide from 1908 to 2009.

Data Cleaning

As we reviewed our data, we noticed the dataset did not contain coordinates for locations of plane accidents. So, we used Microsoft Excel to obtain the appriopiate coordinates for all plane crash locations. Then we used SQL to clean our dataset and create the data file for our visuals.
